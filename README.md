# Tesla Analysis 2025-2026

Python scripts and Matplotlib visualizations analyzing Tesla shareholder deck data and historical trends.

## Key Topics & Visuals
- Cumulative vehicle deliveries vs. active Full Self-Driving (FSD) subscriptions
- Revenue trends
- Stacked deliveries by model
- Annual historical charts
- Energy storage deployment
- Gross & operating margins
- Capex comparisons across companies

## Files Overview
- `cumulative_deliveries_vs_fsd_line.py` — Deliveries vs FSD line chart
- `companies_capex2.py` / `companies_capex.py` — Capex analysis
- `annual_historical_charts.py` — Historical overviews
- PNG files — Generated charts (revenues, deliveries, energy, margins, etc.)
- CSV data files (if any) — Raw Tesla historicals
